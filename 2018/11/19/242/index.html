一、变量

Dart语言中所有变量都是一个对象，每个对象都是一个类的实例。数字类型（numbers）、函数和 null 也是对象。所有对象都继承自 Object 类。

二、常量

使用 final 或 const 声明，表明变量的值不能改变。

1、final

final类型的变量只能设置一次，可以在第一次使用的时候再初始化。

2、const

const是运行时常量（const变量无疑就是final），需要在声明时赋值初始化，在编译时就确定值。
